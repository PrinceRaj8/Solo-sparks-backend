# 🌟 Solo Sparks - Personal Growth Quest System

A self-discovery backend system designed to help users build emotional intelligence through personalized solo quests, multimedia reflections, spark points, and meaningful rewards.

---

## 💡 Key Features

- JWT-based authentication
- Quest engine (daily/weekly suggestions)
- Text, image & audio reflection support via Cloudinary
- Spark Points system for gamification
- Rewards store (boosts, hidden content, exclusives)
- Behavioral analytics (optional extension)

---

## 🛠️ Tech Stack

- **Backend**: Node.js + Express
- **Database**: MongoDB
- **Auth**: JWT (JSON Web Tokens)
- **File Storage**: Cloudinary
- **Middleware**: Custom JWT & upload handlers
